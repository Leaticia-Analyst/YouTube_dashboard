# YouTube_dashboard
# 🚀 YouTube Analytics Dashboard

An interactive data analytics dashboard built with **Streamlit** that allows users to explore YouTube video performance through dynamic visualizations and filtering.

---

## 🔐 Features

- User authentication (login system)
- Interactive dashboard UI
- Category-based filtering
- Key performance metrics (KPIs)
- Data visualization (bar chart & line chart)
- Clean and modern layout

---

## 📊 Dashboard Overview

This app provides:

- 📈 Total Videos, Views, and Likes
- 📊 Views per Video (Bar Chart)
- 📉 Likes per Video (Line Chart)
- 📋 Full dataset preview

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
